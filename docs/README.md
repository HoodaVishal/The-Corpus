# The Corpus — AI & ML Learning Platform

A free, open-source educational platform designed to make Artificial Intelligence 
and Machine Learning accessible to everyone. From Python basics to cutting-edge 
LLMs and AI Agents — everything in one place, structured and easy to follow.

## What You'll Learn

- Python — Core programming fundamentals
- EDA — Exploratory Data Analysis techniques
- Machine Learning — Algorithms and model building
- Deep Learning — Neural networks and architectures
- NLP — Natural Language Processing concepts
- LLM — Large Language Models and how they work
- AI Agents — Building autonomous intelligent systems
- MLOps — Deploying and maintaining ML in production

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## Project Structure

Each subject has its own dedicated folder with a landing page 
and individual lesson pages, making it easy to navigate and contribute.
